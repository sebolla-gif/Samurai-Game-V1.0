Instructions in English:

In this folder, you will find the installer for Code::Blocks 17.12 and the files necessary to include the Allegro library using Windows as the operating system.
The folder labeled 'windows' contains the .dll files that need to be copied into the system32 directory.
The 'include' folder contains the library headers, and the 'lib' folder contains the necessary libraries. 
These last two folders should be included in the configuration of the compiler being used.

Instrucciones en español:

En esta carpeta se encuentra el instalador de Code::Blocks 17.12 y los archivos necesarios para incluir la biblioteca Allegro utilizando Windows como sistema operativo.
La carpeta etiquetada como 'windows' contiene los archivos .dll que deben copiarse en el directorio system32.
La carpeta 'include' contiene los encabezados de la biblioteca, y la carpeta 'lib' contiene las bibliotecas necesarias.
Estas dos últimas carpetas deben incluirse en la configuración del compilador que se esté utilizando.
